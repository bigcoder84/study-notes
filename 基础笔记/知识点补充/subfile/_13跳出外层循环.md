# 跳出外层循环

```java
public class Main {
    public static void main(String[] args) {
        //jump from outer loop
        outer:
        for (int i = 0; i < 5; i++) {
            for (int j = 0; j < 10; j++) {
                if (j == 5)
                    break outer;
                System.out.print("*");
            }
            System.out.print("\r\n");
        }
    }
}
```
